# SW Unit Construction

This folder will host an agentic workflow for unit-level software construction and implementation support.

## Vision
The unit construction workflow will use agents that follow automotive software development standards in general, while remaining configurable for organization-specific coding standards, architecture constraints, and development environments.

## Planned focus areas
- Unit-level design and implementation guidance
- Scaffolding and reusable construction patterns
- Review of coding and safety-related implementation practices
- Support for testability and maintainability
- Customization for team-specific development toolchains

## How to use this folder
Store implementation prompts, templates, scaffolding helpers, and generated unit-level assets here.
Use this workspace to support the disciplined construction of software units in a safety- and quality-conscious manner.
